Git & GitHub Commands Cheatsheet

1. FOR INITIAL PROJECTS or IF YOU ARE JUST TESTING GITHUB

Step 1 — Initialize repository 
git init
Step 2 — Create / edit .gitignore 
(create a file named .gitignore and add patterns)
e.g. node_modules/
     __pycache__/
     .env
Step 3 — Add files to staging 
git add .
Step 4 — Commit 
git commit -m "Initial commit."
Step 5 — Add remote (replace URL) 
git remote add origin https://github.com/username/repo.git
Step 6 — Push to GitHub 
git push -u origin main

2. FOR EXISTING PROJECTS

Step 1 — Check status 
git status
Step 2 — Add files (specific) 
git add file1 file2
Step 3 — Add all changes 
git add .
Step 4 — Commit 
git commit -m "Your descriptive commit message here"
Step 5 — Pull latest (avoid conflicts) 
git pull origin main
Step 6 — Push commits 
git push origin main

3. If files in .gitignore are still being pushed

Step 1 — Remove cached files 
git rm -r --cached .
Step 2 — Re-add files (respect .gitignore) 
git add .
Step 3 — Commit the removal 
git commit -m "Remove ignored files"

4. Getting Code from a Remote Repository

A. First Time (Cloning)

If you don't have the project locally, clone it. This downloads the entire repository to a new folder.

git clone https://github.com/username/repo.git
B. Existing Project (Pulling)

If you have already cloned the project and want to get the latest updates from the remote repository.

# Navigate into your project directory first
cd repo
# Pull the latest changes
git pull origin main
Note 
You only need to `git clone` once. Afterwards, use `git pull` to update your local copy with changes from the remote.

5. To delete the branch "456123branch" (local & remote)

Step 1 — Switch to a safe branch (e.g., main) 
git checkout main
Step 2 — Delete local branch (safe) 
git branch -d 456123branch
Step 3 — Force-delete local branch (if unmerged) 
git branch -D 456123branch
Step 4 — Delete remote branch 
git push origin --delete 456123branch
# or alternative:
git push origin :456123branch
Step 5 — Verify 
git branch       # local
git branch -r    # remote
